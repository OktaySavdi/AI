# Sessions

Manage session history and context.

## Usage

```
/sessions
```

## What it shows

- Recent sessions with task descriptions
- Files modified per session
- Patterns and learnings saved from sessions
- Checkpoints saved with `/checkpoint`

## Session Management

```
/sessions list           # list recent sessions
/sessions show <id>      # show session details
/sessions export <id>    # export session as markdown
```

## Related Commands

- `/checkpoint` — save current task state
- `/learn` — extract patterns from current session
- `/clear` — clear context (free, instant reset)
- `/compact` — summarize context before continuing
