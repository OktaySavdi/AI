# Laravel Security Skill

## CSRF

```php
// Automatically applied to web routes via VerifyCsrfToken middleware
// API routes using Sanctum/JWT are excluded — use token auth instead
Route::middleware('api')->group(function () {
    Route::apiResource('users', UserController::class);
});
```

## SQL Injection Prevention

```php
// WRONG
$users = DB::select("SELECT * FROM users WHERE email = '$email'");

// CORRECT — parameterized
$users = DB::select("SELECT * FROM users WHERE email = ?", [$email]);

// BEST — Eloquent (auto-parameterized)
$users = User::where('email', $email)->get();
```

## XSS Prevention

```blade
{{-- Auto-escaped — SAFE --}}
{{ $user->name }}

{{-- Raw output — ONLY for trusted content --}}
{!! $trustedHtml !!}
```

## Mass Assignment Protection

```php
// Use $fillable (whitelist) — not $guarded = []
protected $fillable = ['email', 'name'];

// NEVER use Model::create($request->all()) without validation
// ALWAYS use $request->validated() or $request->only([...])
$user = User::create($request->validated());
```

## Rate Limiting

```php
// routes/api.php
Route::middleware(['throttle:60,1'])->group(function () {
    Route::apiResource('users', UserController::class);
});

// Custom limiter
RateLimiter::for('login', function (Request $request) {
    return Limit::perMinute(5)->by($request->ip());
});
```

## Secrets (.env)

```env
APP_KEY=base64:...      # generated by artisan key:generate
DB_PASSWORD=            # from environment — never committed
MAIL_PASSWORD=
STRIPE_SECRET=
```

```php
// Access via env() or config()
$secret = config('services.stripe.secret');  // preferred — cached
$debug = env('APP_DEBUG', false);            // only in config files
```

## Security Headers (via middleware)

```php
public function handle(Request $request, Closure $next): Response
{
    $response = $next($request);
    $response->headers->set('X-Content-Type-Options', 'nosniff');
    $response->headers->set('X-Frame-Options', 'DENY');
    $response->headers->set('Referrer-Policy', 'strict-origin-when-cross-origin');
    return $response;
}
```

## Checklist

- [ ] `APP_DEBUG=false` in production
- [ ] `APP_KEY` rotated and not committed
- [ ] All routes authenticated unless explicitly public
- [ ] File uploads validated by MIME type, not extension
- [ ] `$fillable` defined on all models
- [ ] No `DB::raw()` with user input
- [ ] Sanctum/Passport for API authentication
