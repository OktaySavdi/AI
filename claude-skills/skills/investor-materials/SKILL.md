# Investor Materials Skill

## Pitch Deck Structure (10 slides)

```
1. Cover — name, tagline, round size
2. Problem — specific, painful, large
3. Solution — product demo or screenshot
4. Market Size — TAM/SAM/SOM with sources
5. Business Model — how you make money
6. Traction — metrics, growth rate, key customers
7. Competition — positioning map (not "we have no competition")
8. Team — why this team wins
9. Financials — 3-year projections, unit economics
10. The Ask — round size, use of funds, milestones
```

## One-Pager Template

```
[Company Name] — [One-line tagline]

PROBLEM: [1 sentence, specific pain]
SOLUTION: [1 sentence, your product]
TRACTION: [Key metric], [growth rate], [customer names if possible]
MARKET: $[SAM] SAM in [category]
TEAM: [founders, relevant credentials]
ASK: Raising $[amount] to [milestone achieved in 18 months]
Contact: [email]
```

## Financial Model Structure

```
Assumptions Tab:
  - Monthly new customers (by channel)
  - Average contract value / ARPU
  - Churn rate
  - COGS %
  - Headcount plan by quarter

Revenue Tab:
  - MRR/ARR waterfall
  - Customer count
  - Net Revenue Retention

P&L Tab:
  - Revenue, COGS, Gross Margin
  - S&M, R&D, G&A
  - EBITDA, Net Income

Runway Tab:
  - Burn rate, cash on hand
  - Months of runway
```

## Key Metrics to Include

| Metric | Why Investors Care |
|---|---|
| MRR/ARR | Scale and predictability |
| MoM/YoY growth | Trajectory |
| NRR (Net Revenue Retention) | Product-market fit |
| CAC / LTV | Unit economics |
| Gross Margin | Business model quality |
| Burn Multiple | Capital efficiency |

## Investor Memo

A longer-form (5-10 page) narrative version of the pitch deck.
Structure: Executive Summary → Problem → Solution → Market → Traction → Team → Financials → The Ask.
